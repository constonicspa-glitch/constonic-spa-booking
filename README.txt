妳現在只要做：
1. 打開 config.js
2. SUPABASE_URL 貼上 API URL
3. SUPABASE_ANON_KEY 貼上 Publishable key
4. ADMIN_PASSWORD 改成妳自己的後台密碼
5. 存檔
6. 到 https://app.netlify.com/drop
7. 把整個資料夾拖上去
8. 前台網址：netlify網址/
9. 後台網址：netlify網址/admin.html
